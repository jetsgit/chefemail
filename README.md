ChefStep
========

Demo of Using AngularJS for Rails Frondend
------------------------------------------

- Ruby version: 2.2.2p95
- Rails: 4.2.5.2

Gem dependencies
---------------
- gon
- bower-rails
- angular-rails-templates
- responders (for responds to, etc)


App also has example of using Rails API to upload a large query in the background
with Angular.js
